Session 2:
HTML and CSS

HTML = Structure of the website
CSS = Design and Styling of the website

Setup:
1. No need to install HTML or CSS. They are supported by all modern browsers (Chrome, Edge, Firefox, Brave, etc.).
2. Install Live Server (VS Code Extension). It automatically reloads your website whenever you save changes.
3. Create two files:
   - index.html
   - style.css
4. Type ! and press Tab to generate the HTML boilerplate.
5. Link style.css with index.html and start building.
